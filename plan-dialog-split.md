 Plan: AI-split caller/dispatcher chat panel                                                                                                                                                                                                                                      │
│                                                                                                                                                                                                                                                                                  │
│ Context                                                                                                                                                                                                                                                                          │
│                                                                                                                                                                                                                                                                                  │
│ A 9-1-1 call always has exactly two parties: the caller and the dispatcher. Today, Whisper produces a single un-labeled text stream and the frontend renders it in one transcript box (TranscriptPanel.tsx) with keyword highlighting. The new gas-leak-dialog-01 demo scenario contains both voices in one MP3, which makes the single-stream display read like a confusing run-on monologue.                                                                                                                                                                   │
│                                                                                                                                                                                                                                                                                  │
│ Goal: after speech-to-text, an AI call splits the rolling transcript into ordered caller-vs-dispatcher turns, and a chat-style panel displays only the dispatcher's side. The existing transcript box stays unchanged (same text, same kw-highlight). The dispatcher chat lives in the right slot of TranscriptPanel where the "Original Caller Wording" subpanel currently lives.                                                                                                                                                                               │
│                                                                                                                                                                                                                                                                                  │
│ Approach                                                                                                                                                                                                                                                                         │
│                                                                                                                                                                                                                                                                                  │
│ A new dedicated split_transcript() LLM method runs in parallel with the existing extract() call, off the same _schedule_claude trigger (WORD_COUNT_TRIGGER=2, SILENCE_TRIGGER_SEC=0.3). It returns the full turns array each time; the frontend overwrites state. A new transcript_split server event carries the result. The frontend stacks the new dispatcher chat below the existing English transcript inside TranscriptPanel (replacing the "Original Caller Wording" stacked subpanel).                                                           │

│ - CALLER: reports the emergency, often panicked, describes events, asks for help. \                                                                                                                                                                                              │

│ injuries/hazards, emotional language.                                                                                                                                                                                                                                            │

│ RULES                                                                                                                                                                                                                                                                            │
│ - Use the `split_call_turns` tool. Never reply with prose.                                                                                                                                                                                                                       │
│ - Preserve original wording and order. Do NOT translate, summarize, or paraphrase.                                                                                                                                                                                               │
│ - Group consecutive sentences from the same speaker into ONE turn.                                                                                                                                                                                                               │
│ - If only one speaker is present (e.g. live mic before dispatcher speaks), \                                                                                                                                                                                                     │
│ return only that speaker's turns. Empty dispatcher list is valid.                                                                                                                                                                                                                │
│ - The first turn of a 9-1-1 call is almost always the dispatcher's greeting.                                                                                                                                                                                                     │
